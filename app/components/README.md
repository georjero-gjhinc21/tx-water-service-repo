# React Components

All React components for the water service request form.

## Components to Build

- `MultiStepForm.tsx` - Main form container
- `FormStep1Contact.tsx` - Contact information
- `FormStep2Address.tsx` - Address details
- `FormStep3Property.tsx` - Property & services
- `FormStep4Identity.tsx` - Identity & documents
- `FormStep5Review.tsx` - Review & sign
- `SignaturePad.tsx` - Digital signatures
- `FileUpload.tsx` - Document uploads

See main REPOSITORY_STRUCTURE.md for implementation details.
